# Bitrix24 API Assistant — краткое описание

Проект: интеллектуальный чат-бот, предоставляющий разработчикам краткие и точные ответы по документации Bitrix24 API.

Ключевые возможности
- Приём вопросов на естественном языке (текстовой ввод через Telegram).
- Поиск релевантных фрагментов в официальной документации Bitrix24 (RAG-подход).
- Генерация коротких, полезных ответов с использованием Yandex Cloud LLM (при наличии ключа).

База знаний
Официальная документация Bitrix24: https://apidocs.bitrix24.ru/

Структура репозитория
- `bot.py` — основной интерфейс Telegram (aiogram).
- `assistant.py` — обёртка для работы с Yandex Generation API (с локальным fallback).
- `bitrix24_rag.py` — простой извлекатель контекста из документации (скрейпинг + ранжирование).
- `utils.py` — вспомогательные утилиты для обработки текста.
- `config.example.env` — пример переменных окружения.
- `requirements.txt` — зависимости проекта.

Быстрый старт (локально)
1. Скопировать пример конфигурации и заполнить переменные:

```bash
cp config.example.env .env
# Отредактируйте .env и укажите TELEGRAM_TOKEN. При наличии YANDEX_API_KEY — заполните тоже.
```

2. Установить зависимости и запустить бота:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python bot.py
```

Примечания
- Для полноценных ответов через LLM заполните `YANDEX_API_KEY` и `YANDEX_FOLDER_ID` в `.env`.
- `bitrix24_rag.py` реализует базовый RAG-подход через скрейпинг; для продакшна рекомендуется индексирование и использование векторной БД.
- Я не включаю в репозиторий приватные ключи или токены — используйте `.env`.

Описание проекта

Проект предполагает создание интеллектуального чат-бота, работающего через Yandex Assistant, который будет отвечать на вопросы разработчиков по документации API Bitrix24. Бот должен понимать естественный язык, извлекать информацию из технической документации и выдавать точные, краткие и полезные ответы.

Пользователь (разработчик) задаёт вопрос голосом или текстом, а бот, используя обработку запроса и доступ к базе знаний, формирует корректный ответ, основываясь на документации Bitrix24 API.


Ссылка на базу знаний: https://apidocs.bitrix24.ru/.
